== Description ==
A simple, lightweight, fully Responsive WordPress Team display plugin to create & manage your Team page easily with three different column layouts and 20 different color sets.

It has 20 predefined color schemes and three different layouts and option to add social links. Admin can easily make the status inactive of a staff to hide it from the front end. Also can easily change order of the staffs from the order meta box. Just use the shortcode 1[wp-super-team]` in the editor.


== Installation ==
1. Upload `wp-super-team` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Use shortcode `[wp-super-team]` to show team in page/post editor
